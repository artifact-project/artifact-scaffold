Artifact Scaffold
-----------------
Быстрый страт проекта на Exility + Elastin.

 - `npm run dev` — запуск dev режима
 - `npm start` — production версия
   - `--analyzer` — анализатор бандлов


### Features

 - Isomorphic
 - CSS in JS
 - Hot reloading
 - Reactive data-flow


### Development

 - `npm i`
 - `npm test`, [code coverage](./coverage/lcov-report/index.html)
